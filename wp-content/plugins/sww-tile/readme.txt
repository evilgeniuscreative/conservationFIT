=== Tiles ===

Adds a shortcode to make pages of little tiles easy.

== Description ==

Add the 'sww-tile' short code to wrap your text:

   [sww-tile]Click here to enter admin page[/sww-tile]

Optional "height=" and "width="" parameters can override the default.

== Installation ==

1. Unzip the plugin archive to the plugins directory.
2. Activate the plugin.
